# Recipe Sharing Platform

## Description

The Recipe Sharing Platform is a web application that allows users to share their culinary recipes, discover new recipes, and manage their favorite recipes. Users can add recipes with details such as ingredients, instructions, and an optional image.

## Features

- **Add Recipes**: Input and save recipes with title, ingredients, instructions, and an optional image.
- **View Recipes**: List all added recipes with their details and images.
- **User-Friendly Interface**: Simple and intuitive design for easy interaction.

## Technologies

- **Frontend**: HTML, CSS, JavaScript
